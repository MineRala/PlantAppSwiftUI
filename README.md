# PlantApp

# Pages
- GetStarted Screen
- Onboarding1 Screen
- Onboarding2 Screen
- Paywall Screen
- Home Screen

# Compatibility
- Requires iOS 17.0 or later. 
- Compatible with portrait-oriented iPhone.

# Architecture
- MVVM architecture was used in this project.

# Features
- Responsive and Pixel-Perfect UI
- Used Free API for data.

# Technologies
- SwiftUI
- MVVM
- Async/Await
- Error Handling
- Tabbar
